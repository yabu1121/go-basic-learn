## 🚀 使い方

### 1. 問題を解く
各ファイル（`01.go` 〜 `08.go`）を開いて、問題の要件を読んでください。

### 2. コードを書く
各`problemN()`関数の中にコードを書いてください。

### 3. 実行する
```bash
go run 01.go
go run 02.go
# ... 以下同様
```

### 4. 出力を確認
期待される出力例と比較して、正しく動作しているか確認してください。

## 💡 学習のコツ

1. **まず自分で考える**: すぐに答えを見ずに、まず自分で実装してみましょう
2. **元のファイルを参照**: 詰まったら対応する学習ファイル（`01_hello.go`等）を見返しましょう
3. **実行して確認**: こまめに実行して、動作を確認しながら進めましょう
4. **エラーを恐れない**: エラーメッセージは学習の材料です
5. **応用してみる**: 問題を解いたら、自分なりにアレンジしてみましょう
